# supiano908-dev
Dépôt réservé aux devs pour récupérer le code source ainsi que d'autres infos à Propos de Supiano908 (https://lepine908.github.io/supiano.html)

Supiano908 est un projet TurboWarp français sous licence MIT. Appuyez sur le drapeau vert pour redémarrer et sur « start » pour commencer. Vous découvrirez le monde fascinant de la musique, dévoilé et lancé pour la première fois au Collège Saint-Exupéry bientôt ce printemps 2026, accessible en ligne. Une version Française est en cours de développement. Créé par lepine908 de 908 Studios. Codé avec le moteur TurboWarp sous environnement Zorin (Windows et Linux). Réalisé dans le cadre d’un projet petit pour tester les jeux en ligne.

Accédez-y via ce lien : https://lepine908.github.io/supiano.html Recherche par Adrocode, développemnt musical par Adrocode. Présenté par lepine908 pour 908 stduios. Certains sons proviennent de Universal Soundbank International. @lepine908

Voici un projet pour faire de la musique numériquement ! Supiano908p.sb3 contient le fichier d'application Scratch (.sb3). Vous pouvez l'ouvrir avec TurboWarp en ligne et tester notre projet(ATTENTION:ne contient pas les éléments cloud pour des raisons de sécurité(projet open source)) ! Supiano908.html est un fichier HTML compilé par TurboWarp qui contient le fichier .sb3 pour une ouverture facile dans un navigateur. Contactez les développeurs sur https://lepine908.github.io. Amusez-vous bien !
